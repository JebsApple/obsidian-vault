# Suwayomi & Mihon — Updates 2026

## Suwayomi Server v2.2
- Antes Tachidesk, ahora Suwayomi
- Corre en Windows, macOS, Linux, Docker
- Web UI + OPDS + API
- Extensiones compatibles con Mihon/Tachiyomi
- MyAnimeList + AniList integration
- AUR package: `suwayomi-server-bin`

## Alternativas
- **Komga** — servidor de comics/manga, multi-usuario
- **Kavita** — UI linda, soporta comics + ebooks + manga
- **Mango** — servidor minimalista
