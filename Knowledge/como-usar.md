# Cómo usar este vault

## Para opencode
El servidor MCP está configurado en \opencode.jsonc\. Usa herramientas como:
- \search-vault\ - Buscar notas
- \ead-note\ - Leer notas
- \create-note\ - Crear nuevas notas de memoria

## Para Claude Code
Configurado en \~/.claude/settings.json\. Mismas herramientas disponibles.

## Buenas prácticas
- Crea notas con frontmatter (tags, tipo)
- Usa \Knowledge/\ para conceptos reutilizables
- Usa \Projects/\ para contexto específico de proyectos
