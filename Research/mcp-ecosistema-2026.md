# MCP Ecosystem 2026

Estado del Model Context Protocol a junio 2026.

## Cifras
- ~90K+ MCP servers catalogados (~8K activos validados)
- MCP donado a Linux Foundation (Agentic AI Foundation) dic 2025
- 8M+ descargas semanales de SDKs MCP

## Empresas con soporte nativo
Stripe, GitHub, Postgres, Slack, Notion, Google, OpenAI, Microsoft

## Tendencias
- **Streamable HTTP Transport** — nuevo estándar en junio 2025
- **OAuth 2.1 + PKCE** para auth
- MCP como "USB-C de AI" — estándar de facto
- Cursor, Claude Desktop, ChatGPT, VS Code todos con soporte

## Relevancia
- [[opencode]] + Claude Code ya usan MCP
- Stripe tiene MCP server público — útil si se manejan pagos
- Se puede exponer cualquier API propia como MCP server
