## Diapositiva 3

Diagrama de Despliegue
Representa la asignación de artefactos de software a máquinas físicas (nodos).
Muestra los enlaces de comunicación entre los nodos.
Un nodo puede representar un dispositivo de hardware donde se ejecuta el software u otro elemento físico más sencillo como una impresora.
Puede usarse un estereotipo para diferenciar los tipos de nodo <<  >>

## Diapositiva 4

Nodo
Un nodo tiene la forma de un cubo o prisma rectangular  y generalmente representa un  dispositivo de hardware donde se aloja el software o puede representar un ambiente de ejecución.
Dentro de un nodo se ubican los artefactos.

## Diapositiva 5

Artefacto
Un artefacto representa un producto de software.

## Diapositiva 6

Componente
El concepto de componente se  refiere a un componente de software reutilizable
En la industria del automóvil, un componente es, por ejemplo:
Un motor.
Una puerta.
Un asiento

## Diapositiva 7

Relación de Manifestación
Una manifestación es una implementación concreta de un artefacto.
Un artefacto se puede manifestar en un componente o un paquete
