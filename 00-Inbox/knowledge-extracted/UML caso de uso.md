## Diapositiva 3

¿Qué es UML?
UML significa Unified Modeling Language (Lenguaje Unificado de Modelado)
Es un lenguaje gráfico para visualizar, especificar, construir y documentar sistemas de software
No es un lenguaje de programación, sino un lenguaje de modelado
Está estandarizado por el OMG (Object Management Group)

## Diapositiva 4

¿Para qué se usa UML?
Comprender y comunicar ideas sobre el sistema entre analistas, desarrolladores y clientes.
Diseñar la arquitectura del sistema antes de programar.
Documentar el sistema para su mantenimiento y evolución.
Detectar errores conceptuales tempranos.

## Diapositiva 5

Beneficios del modelado con UML
Mejora la comunicación entre los equipos.
Facilita el análisis y el diseño del sistema.
Permite dividir el sistema en partes comprensibles.
Apoya la reutilización de modelos en otros proyectos.
Sirve como documentación del software.

## Diapositiva 7

Casos de uso
El Diagrama debe seguir la notación para casos de uso establecida por UML
Actores.
Casos de Uso.
Relaciones.
Las relaciones de Actores con casos de usos se denominan “Asociaciones”.
Las relaciones entre casos de uso se denominan “Generalizaciones” y pueden ser de dos tipos, de uso (Uses) o de herencia (Extends).
Según el número de casos de uso se puede usar un diagrama, o varios según los módulos o funcionalidad.

## Diapositiva 11

Sistema de Biblioteca Online
Una biblioteca universitaria quiere implementar un sistema web donde los estudiantes puedan buscar libros, reservarlos y devolverlos. El personal bibliotecario podrá registrar nuevos libros, actualizar su información y ver el estado de las reservas. Solo los usuarios registrados podrán usar el sistema.

## Diapositiva 12

Sistema de Biblioteca Online
Requerimientos funcionales
El estudiante debe poder registrarse e iniciar sesión.
El estudiante puede buscar libros por título, autor o categoría.
El estudiante puede reservar un libro disponible.
El estudiante puede cancelar una reserva antes de recoger el libro.
El bibliotecario puede registrar nuevos libros en el sistema.
El bibliotecario puede modificar o eliminar libros.
El bibliotecario puede ver todas las reservas activas.

Sistema de Biblioteca Online
