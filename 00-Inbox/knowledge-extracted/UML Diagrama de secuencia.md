## Diapositiva 3

Diagrama de secuencia
Un diagrama de secuencia es un tipo de diagrama UML que muestra cómo los objetos interactúan entre sí a lo largo del tiempo para llevar a cabo un proceso o un caso de uso específico.

## Diapositiva 4

¿Para qué se usa?
Para visualizar la secuencia de mensajes (llamadas de métodos, respuestas, etc.) entre objetos o componentes.
Para detallar el flujo de ejecución paso a paso de un escenario.
Para entender el comportamiento dinámico del sistema.
