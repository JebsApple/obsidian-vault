# Ramas en español con nomenclatura estricta

## Lo que aprendí
Todas las ramas se nombran en español. Nomenclatura: `S3-HUXX-TXX-descripcion-breve` o `SLT-descripcion`. Confirmar con el usuario antes de renombrar ramas que no creaste tú mismo. Las ramas viejas se respaldan como tags `archive/` antes de eliminar.

## Contexto
2026-06-23 en MiNegocio (HU01-T05). Se renombraron ramas de `feature/huXX-tXX-*` a `S3-HUXX-TXX-*`.

## Aplicación
Establecer la convención de nombres de ramas al inicio del proyecto y documentarla. No renombrar ramas de otros sin confirmar.
