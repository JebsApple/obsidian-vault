# Iglesia Project — Jesus Reigns Under Worship

**Objetivo:** Web moderna y minimalista para la iglesia "Jesus Reigns Under Worship" que sirva como hub central de información, mantenido por persona no técnica.

**Estado actual:** Solo redes sociales (FB, IG, WhatsApp, YouTube), sin web existente.

**Stack:** Astro + Tailwind CSS, Decap CMS (CRUD para no-técnicos), Vercel (hosting gratis), Netlify Forms (buzón de oración), Cloudinary (galería).

**Estilo:** Celeste, blanco, negro, dorado/trigo. Minimalista, moderno, familiar. Nubes y trigo como elementos decorativos.

**Relación con otros proyectos:**
- App Luciernaga = independiente, sin integración
- Scan-tracker = independiente
